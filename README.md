## github pages 使い方

1. GitHubでリポジトリを作成する

2. 作成したリポジトリをcloneする

   `git clone <リポジトリのURL>`

3. cloneしたリポジトリにREADME.mdを作成し、pushする（origin/master）

4. 「gh-pages」ブランチを作成する

5. gh-pagesブランチでindex.htmlを作成し、pushする（origin/gh-pages）

6. URLにアクセスする

   `https://<GitHubのアカウント名>.github.io/<リポジトリ名>`
